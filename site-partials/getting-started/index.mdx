# react-trailer <VersionTag version={version} name={name} />

react-trailer is a wrapper around HTML5 video API that allows you to render videos in your React application.

## Installation

First of all, install the following **peer dependencies**:

<Code language="bash">{`npm i ${Object.keys(peerDependencies).join(' ')}`}</Code>

To add the library to your project you can either use a CDN service:

<Code language="html">{`<script src="https://unpkg.com/${name}@${version}/index.js"></script>`}</Code>

Or npm to install the library from npm registry:

<Code language="bash">{`npm i --save ${name}`}</Code>

Or if you prefer yarn:

<Code language="bash">{`yarn add ${name}`}</Code>
